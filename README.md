# FormReturn License Generator
This generator is for past FormReturn customers who need to generate a license key file to unlock version 1.7.4 or lower.

## Requirements
OpenJDK 8 or Java 8.

## Running
Download the frlicense.jar distribution, then from a terminal or command prompt, enter the following command:
```
java -jar frlicense.jar
```

## Building
```
mvn clean package
```
